# 🤖 Automação de Cadastro e Compra Online com BotCity

## 📌 Descrição
Este projeto consiste no desenvolvimento de um bot de automação desktop utilizando o **BotCity Framework Core** com técnicas de **visão computacional**.

O robô foi desenvolvido para automatizar o fluxo completo de compra no site **Automation Exercise**, simulando a interação humana com a interface gráfica do sistema.

O processo automatizado contempla:
- acesso ao site
- cadastro de novo usuário
- preenchimento dos dados pessoais
- navegação até o carrinho
- continuação da compra
- preenchimento dos dados do cartão
- confirmação do pedido
- acesso ao comprovante final da compra

---

## 🎯 Objetivo
Automatizar tarefas repetitivas de cadastro e compra online utilizando **RPA com Python e BotCity Studio**, reduzindo tempo de execução e minimizando erros operacionais.

---

## ⚙️ Tecnologias utilizadas
- Python
- BotCity Framework Core
- BotCity Studio
- Visão Computacional
- Automação Desktop
- BotCity Maestro SDK (opcional)
